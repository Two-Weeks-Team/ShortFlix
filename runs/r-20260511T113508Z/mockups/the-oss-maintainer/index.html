Quickstart

$ pip install openseowriter
$ export GOOGLE_API_KEY=...

$ osw write \
    --topic "한국 1인 사업자 세무 신고 가이드" \
    --target-keywords "종합소득세,간이과세,홈택스" \
    --tone trustworthy \
    --length 2400 \
    --publish wordpress://...

How it works (4 agents)

① KeywordAgent — Vertex AI Search + Google Trends (MCP) → 검색량·CPC·난이도 분석
② OutlineAgent — 경쟁 상위 10개 글 분석 → 7-section 구조 도출
③ DraftAgent (Gemini Pro) — 한국어/영어 콘텐츠 작성, 출처 인용
④ FactCheckerAgent (A2A) — 외부 fact-checker partner agent와 cross-validation
두 agent가 disagree 하면 user에게 highlight 표시.
