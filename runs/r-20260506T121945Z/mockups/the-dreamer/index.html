Why this is grounded, not science fiction

  • ADK is composable today. A 4-agent topology in 2026 becomes a 4×N agent topology when N households join. Same orchestrator interface.
  • MCP is the gossip protocol. Tool calls between agents are already structured; an MCP-mesh between agents (not just to APIs) is the next obvious step.
  • Gemini private inference + Vertex AI grounding means embeddings can be exchanged without leaking user histories — the differential-privacy budget is the only knob we add.
  • Cloud Run → GKE is the scaling story we already plan to tell judges in the live demo.
